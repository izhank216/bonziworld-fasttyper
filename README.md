# BonziWORLD FastTyper
With this extension, You can type fast in BonziWORLD!

## Installation
For Tampermonkey (for firefox)
- 1. install the [tampermonkey extension](https://www.tampermonkey.net/)
- 2. create a "new user script".
- 3. copy and paste the code from `tampermonkey/script.user.js`
- 4. save (ctrl+s)
- 5. done!

For Chrome/Chromium
- 1. go to releases (https://github.com/izhank216/bonziworld-fasttyper/releases/tag/1.0)
- 2. download chrome.zip from it
- 3. extract the chrome.zip with a extractor like 7-zip or winrar.
- 4. then open chrome and go to "chrome://extensions"
- 5. turn on developer mode (top right)
- 6. click load unpacked
- 7. select the chrome folder.
- 8. done!
